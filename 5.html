yoel html5 2d game engine programming tut 5

sound

in this tutorial i will show how to use sound i your game.

the finished project is in file 5)sound

first we will create a new sound obgect in the entitys class constructor. 


				////////extanded_entity///////////

				var extanded_entity = function(x,y,z,yimg,type)
				{
					this.type = 'extanded_entity'; 

					//extand entity
					y_entity.call(this,x,y,z,3,yimg);
					
					//y_sound(sound,volume,loop);
					
					this.test_sound = new y_sound("sound/laser.wav",.12,false);
					
					this.did_init = false;

				}//end extanded_entity

the class y_sound gets 3 arguments:sound,volume,loop. sound: a link to the sound file.
volume:the sounds volume from 0 to 1.
loop:a boolean that determines if the sound will loop when it finesses playing.


				y_sound(sound,volume,loop);

next we will initiate the sound class in our init method: 


				extanded_entity_p.init = function()
				{
					if(!this.did_init)
					{
						ytrace("did init");
							//init click handler
						
						 this.test_sound.init();
						
						this.did_init = true;
					
						
					}
				}//end init

then we will create a new method called play sound: 


				extanded_entity_p.play_sound = function()
				{
					
						//play sound on click space
						if(y_input2_p.key_down[y_key.SPACE])
						{
							this.test_sound.play();
						}

					
				}//end move

the sound will play when space is clicked. there is also a stop method: 


				this.test_sound.stop();

finely we will add the method to the update method: 


				extanded_entity_p.update = function()
				{
					//do stuff when class created
					this.init();
					

					this.play_sound();
					
					
					//call parent update func
					y_entity_p.update.call(this); 

				}//end update
