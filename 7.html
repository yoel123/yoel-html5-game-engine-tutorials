timers

in this tutorial i will show how to use timers in your game
we will make an entity move to the left every second.

the finished project is in file 7)timers

first we will will create a timer object in our entity class constructor


				////////extanded_entity///////////

				var extanded_entity = function(x,y,z,yimg)
				{
					this.type = 'extanded_entity'; 

					//extand entity
					y_entity.call(this,x,y,z,3,yimg);
					
					this.test_timer =  new y_timer(1);
					
					
					this.did_init = false;

				}//end extanded_entity

the only argument the timer obgect gets is the duration,in this case
approximately one second.
no we will create a new method and call it timer. 


				extanded_entity_p.timer = function()
				{
					
				}//end timer

we will also add it to update method: 


				extanded_entity_p.update = function()
				{
					//do stuff when class created
					this.init();
					

					this.timer();
					
					
					//call parent update func
					y_entity_p.update.call(this); 

				}//end update

we will update the time by calling test_timer.update();, so it will update
every time the entity updates. if the timer is finished (meaning a second had passed),move the entity left.
if the entity reaches the game screen edgy reset its x pos.
also we will reset the timer after its finished so it will start again. 


				extanded_entity_p.timer = function()
				{
					
					//update timer
					this.test_timer.update();
					
					//if timer finished
					if(this.test_timer.finished )
					{
						//move left
						y_entity_p.move_by.call(this,this.speed,0,0);
						//reset timer
						this.test_timer.reset()
					}
					
					//reset position
					if(this.x>300){this.x=0;}
					

					
				}//end timer

thats how you use a timer,its a very impotent concept i use timers in almost every game i make.
